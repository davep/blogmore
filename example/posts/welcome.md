# Welcome to Blogmore!

This is an example blog post demonstrating the capabilities of **Blogmore**, a blog-oriented static site generation engine.

## Features

Blogmore supports all the standard markdown features:

- **Bold text** and *italic text*
- Lists (ordered and unordered)
- Code blocks
- Links and images
- And much more!

## Code Example

Here's a Python code snippet:

```python
def hello_world():
    print("Hello from Blogmore!")

hello_world()
```

## Conclusion

With Blogmore, creating a blog is as simple as writing markdown files. Enjoy!

{# Reusable post summary macro. Renders a single post summary article element. Used on every page that lists posts: the home page index, tag pages, category pages and date-based archive pages. Centralising the markup here means that a single edit to this file changes the way posts are presented in every listing context. Parameters: post: The Post object to render. include_category (bool): Whether to render the post's category link. Defaults to true. Pass false on the category page, where the category is already implied by the page's own heading. Context variables used (passed via ``with context``): with_read_time (bool): Whether to show the estimated reading time. category_dir (str): URL path segment for category pages. tag_dir (str): URL path segment for tag pages. #} {% macro post_summary(post, include_category=true) %}

{{ post.title }}

{% if post.date %} {{ post.date|format_date }} {% endif %} {% if with_read_time %}
{{ post.reading_time }} min read
{% endif %} {% if include_category and post.category %}
{{ post.category }}
{% endif %}
{{ post.html_content|safe }}
{% if post.tags %}
{% for tag, safe_tag in post.sorted_tag_pairs() %} {{ tag }} {% endfor %}
{% endif %}
{% endmacro %}
